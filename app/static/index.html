About SignalStack

PostgreSQL Search

Weighted TSVECTOR full-text search with GIN indexing, ranking, phrase search, filtering, and pagination.

Semantic AI Retrieval

Local sentence-transformer embeddings enable semantic similarity search and related-article recommendations.

Hybrid Ranking

Combines classical keyword retrieval with embedding similarity scoring for improved relevance.

Machine Learning

scikit-learn classification pipeline with persisted vectorizers, models, confidence scoring, and live inference.

Offline AI Architecture

Embedding models are loaded locally to support air-gapped and offline deployment scenarios.

FastAPI Platform

Modular service-layer architecture exposing AI-powered REST endpoints and static frontend integration.
